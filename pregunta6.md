Parte 6: Relación entre XML y web
Responder:
    1. ¿Qué información aparece tanto en el XML como en la web?
        La informacion que se toma mediante el JS y se mapea en etiquetas HTML se saca de el XML, es el contenido de el banner superior, el listado de actividades, las fichas de cada actividad y los datos de contacto


    2. ¿Por qué podría ser útil guardar los datos en XML?
        Para generar un archivo plano estrucutrado con el cual poder utilizar mediante JS para popular la pagina.

    3. ¿Qué comprueba la DTD?
        Valida la estructura y el orden de los elementos.
        Valida la cantidad de elementos asi como la obligatoriedad de los atributos para asegurarvalidez del coumento.

    4. ¿Qué ocurre si cambio el orden de los elementos?
        si el orden de los elementos establecido en el DTD es modificado en el XML se pruduce un error de Valicadion.

    5. ¿Qué ventaja tiene usar atributos como id o tipo?

        Con el ID se peuden realizar busquedas mas precisas, asi como verifica que cada ID sea unico en su nodo.

        Permite definir categorias de datos en los XML facilitando su filtrado
